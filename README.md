## 🌦️ **README – Weather App Prototype**

```markdown
# Weather Dashboard (React + OpenWeather API)

A personal project to build a responsive weather dashboard using **React** and the **OpenWeatherMap API**.

## 🚀 Overview

Displays current weather data and temperature in Celsius or Fahrenheit. Designed to practice API integration, component structure, and responsive layouts.

## 🧰 Tech Stack

- **Frontend:** React, Vite
- **API:** OpenWeatherMap (JSON data fetch)
- **Styling:** CSS Grid / Flexbox

## ⚙️ Features Implemented

- Fetch weather data by city name.
- Toggle between Celsius and Fahrenheit.
- Responsive design tested in Chrome/Firefox dev tools.

## 🧩 In Progress

- Adding 5-day forecast view.
- Implementing dynamic background images based on weather conditions.
- Improving UI polish and error messages.

## 💡 What I Learned

- How to call APIs asynchronously with `fetch()` and `async/await`.
- Managing component state and props in React.
- Debugging CORS and environment variable issues.
```
